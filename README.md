# 👋 Hi, I'm Jo Bendik

Gameplay Developer building browser-based games, real-time interactive systems, and technically ambitious game projects.

I work across TypeScript, JavaScript, web game development, multiplayer prototypes, AI behavior, custom tooling, and low-level systems. My projects range from tactical shooters and roguelike deckbuilders to arcade prototypes, multiplayer experiments, virtual machines, compilers, and custom computer architecture.

I care about responsive controls, readable architecture, strong player feedback, and systems that are both technically interesting and enjoyable to play.

## 🚀 Selected Projects

### 🔫 [BreachVector](https://github.com/jobendik/BreachVector)

A Phaser 3 + TypeScript tactical top-down shooter with stealth, enemy AI, destructible cover, weapons, objectives, and extraction-based missions.

### 🛰️ [LastSignal](https://github.com/jobendik/LastSignal)

A tactical sci-fi roguelite tower defense game built with TypeScript, Vite, and Canvas 2D. Focused on wave defense, hostile anomalies, procedural audio, progression, and neon CRT-inspired presentation.

### 🃏 [Veilbreach](https://github.com/jobendik/Veilbreach)

A dark sci-fi roguelike deckbuilder prototype with card combat, relic systems, encounters, progression, and data-driven architecture.

### ⚔️ [EchoesOfTheGrid](https://github.com/jobendik/EchoesOfTheGrid)

A grid-based tactical deckbuilder where the player commands a squad of operators through combat, events, rest nodes, forge nodes, and boss encounters.

### 🧪 [GameDesignLab](https://github.com/jobendik/GameDesignLab)

A collection of 34 browser-based game prototypes exploring different genres, mechanics, controls, game feel, and future full-game concepts.

### 🧠 [Nexa](https://github.com/jobendik/Nexa)

A custom 16-bit computer platform including instruction set architecture, assembler, virtual machine, compiler, and operating system experiments.
